# 智能皮肤检测小程序 - 功能速览

## 🎯 一句话总结

这是一个基于 AI 的微信小程序，通过拍照检测皮肤状态，提供个性化护肤方案，支持历史记录对比和产品推荐。

---

## 📱 核心功能（5 大模块）

### 1. 用户系统 👤
- 微信一键登录
- 个人中心（头像、昵称）
- 检测次数统计
- 个人资料编辑

### 2. 智能皮肤检测 📷
- **5 秒自动扫描**：人脸识别 + 扫描动画 + 自动拍照
- **AI 分析**：豆包视觉模型分析 9 项指标
- **检测报告**：综合评分 + 皮肤类型 + 各项指标详情
- **护肤建议**：针对性建议

### 3. 产品推荐 🛍️
- 根据皮肤类型推荐产品
- 根据皮肤问题推荐产品
- 个性化护肤方案

### 4. 历史记录 📋
- **时间线视图**：按时间查看所有检测记录
- **日历视图**：按月查看检测记录
- **记录详情**：查看完整分析结果
- **历史对比**：选择 2 条记录对比皮肤改善情况
- **删除记录**：管理历史数据

### 5. NFC 碰一碰 📡
- NFC 标签跳转小程序
- 美容店导览、产品展示

---

## 🔬 AI 分析的 9 项指标

| 指标 | 说明 |
|------|------|
| 💧 水分 | 皮肤水分含量 |
| 🛢️ 油分 | 皮肤油分 |
| 🌡️ 敏感度 | 皮肤敏感程度 |
| 🔴 痘痘 | 痘痘检测 |
| 🌀 皱纹 | 皱纹检测 |
| 🟤 色斑 | 色斑检测 |
| ⚫ 毛孔 | 毛孔大小 |
| ⚫ 黑头 | 黑头检测 |
| 🏷️ 皮肤类型 | 干性/油性/混合性/中性 |

```
首页 (Landing)
├─ 开始检测 → 相机页面

└─ 开始检测 → 分析页面 → 结果页面 → 推荐页面

历史列表
├─ 时间线视图
├─ 日历视图

## 💡 亮点功能

1. **5 秒自动扫描**：无需手动拍照，自动识别并拍摄
2. **智能分析**：豆包视觉模型，精准分析 9 项指标
3. **历史对比**：追踪皮肤改善情况
4. **日历视图**：直观了解检测频率
5. **个性化推荐**：根据分析结果推荐产品
6. **NFC 跳转**：碰一碰打开小程序

## 🚀 技术栈

- **前端**：Taro 4.x + React 18 + Tailwind CSS
- **后端**：NestJS + 云托管
- **AI**：豆包视觉模型（doubao-vision）
- **存储**：微信云数据库 + 云存储

## 🎯 用户使用流程

```
1. 打开小程序 → 登录
2. 点击"开始检测"
3. 等待 5 秒扫描（自动拍照）
4. 查看 AI 分析结果
5. 查看产品推荐
6. 在"历史"中查看之前的记录
