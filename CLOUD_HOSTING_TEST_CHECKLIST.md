## ⚠️ 常见问题排查

### 问题 1：小程序登录失败

**可能原因：**
1. WECHAT_APP_SECRET 配置错误
2. 小程序 AppID 不匹配
3. 网络请求域名未在白名单中

**排查方法：**
1. 检查云托管环境变量中的 WECHAT_APP_SECRET
2. 检查 config/index.ts 中的 TARO_APP_WEAPP_APPID
3. 查看云托管日志中的错误信息
4. 确认小程序后台的服务器域名配置

### 问题 2：图片上传失败

**可能原因：**
1. 云存储配置错误
2. 图片大小超限
3. 网络超时

**排查方法：**
1. 检查 COZE_SUPABASE_URL 和 COZE_SUPABASE_ANON_KEY
2. 查看云托管日志中的 CloudStorageService 初始化日志
3. 检查图片大小是否超限（建议 < 10MB）

### 问题 3：AI 分析失败

**可能原因：**
1. COZE_API_KEY 配置错误
2. Coze API 连接超时
3. 图片格式不支持

**排查方法：**
1. 检查 COZE_API_KEY、COZE_MODEL、COZE_API_BASE
2. 查看 CloudStorageService 初始化日志
3. 确认图片格式（支持 JPG、PNG）
4. 查看云托管日志中的错误详情

### 问题 4：历史记录为空

**可能原因：**
1. Supabase 连接失败
2. 数据库表未创建
3. 数据读取权限问题

**排查方法：**
1. 检查 Supabase URL 和 Key
2. 查看云托管日志中的 Supabase 初始化日志
3. 确认数据库表是否创建（users、skin_history）
