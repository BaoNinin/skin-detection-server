🎯 NFC 配置工具

快速配置 NFC 标签，实现跳转微信小程序

1 配置网页 URL
💡 这是部署后网页的完整访问地址，请替换为你实际的服务地址
2 配置小程序 URL Scheme
💡 在微信公众平台 → 开发管理 → 开发设置 → URL Scheme 中生成
3 生成配置代码
📄 nfc-redirect.html 配置代码
4 写入 NFC 标签
💡 使用 Android 手机写入（推荐）
1. 下载 NFC Tools App
2. 选择"写入" → "添加记录" → "URI"
3. 输入上面的网页 URL
4. 靠近 NFC 标签写入
? 使用说明
完整配置流程：

1. 部署网页到服务器
2. 获取小程序 URL Scheme
3. 在此工具中生成配置代码
4. 将配置代码复制到 nfc-redirect.html
5. 使用 NFC Tools 写入网页 URL 到标签

详细文档：查看 NFC_CONFIG.md
