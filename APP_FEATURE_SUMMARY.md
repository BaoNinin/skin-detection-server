# 智能皮肤检测小程序 - 完整功能清单

## 📱 项目概述

**应用名称**：智能皮肤检测微信小程序

**核心功能**：基于 AI 的面部皮肤检测与分析系统，提供个性化的护肤方案

**技术栈**：
- 前端：Taro 4.x + React 18 + Tailwind CSS 4.x
- 后端：NestJS + 微信云托管
- AI：豆包视觉模型（doubao-vision）
- 存储：微信云数据库 + 云存储

---

## 🎯 核心功能模块

### 1. 用户系统

#### 1.1 用户登录
- **微信授权登录**
  - 使用微信小程序原生登录接口
  - 自动获取用户 OpenID
  - 新用户自动注册，老用户直接登录
  - 本地存储用户信息（userId、userInfo）

- **手机号登录**（可选）
  - 支持手机号快速登录
  - 获取用户手机号信息

#### 1.2 个人中心
- **用户信息展示**
  - 头像（支持自定义）
  - 昵称（支持修改）
  - 检测次数统计
  - 注册时间显示

- **个人资料编辑**
  - 修改昵称
  - 更换头像
  - 保存个人信息

- **账户管理**
  - 退出登录
  - 清除本地缓存
  - 显示登录状态

### 2. 皮肤检测功能

#### 2.1 拍照检测
- **智能相机**
  - 前后摄像头切换
  - 闪光灯控制（关闭/开启/常亮）
  - 高清拍照模式
  - 相机就绪检测

- **5秒自动扫描流程**
  - 启动检测后自动开始 5 秒扫描
  - 面部轮廓实时显示
  - 扫描动画效果（12 个扫描点椭圆形分布）
  - 扫描数据实时展示：
    - 面部轮廓识别进度
    - 皮肤特征提取进度
    - 水分检测进度
    - 纹理分析进度
  - 扫描完成后自动拍照

- **人脸识别辅助**
  - 标准椭圆面部轮廓框
  - 扫描点完美匹配轮廓
  - 视觉引导用户对准面部

#### 2.2 AI 皮肤分析
- **豆包视觉模型分析**
  - 上传皮肤照片到云托管
  - 调用豆包视觉模型 API
  - 多维度皮肤分析：
    - 皮肤类型识别（干性、油性、混合性、中性）
    - 水分含量分析（0-100 分）
    - 油分分析（0-100 分）
    - 敏感度分析（0-100 分）
    - 痘痘检测（0-100 分）
    - 皱纹检测（0-100 分）
    - 色斑检测（0-100 分）
    - 毛孔检测（0-100 分）
    - 黑头检测（0-100 分）

- **芯片激活动画**
  - 模拟扫描识别过程
  - 强化"扫描识别"而非"拍照识别"的用户体验
  - 视觉反馈：数据传输、分析中、结果生成

#### 2.3 检测结果展示
- **综合评分**
  - 根据各项指标计算综合评分（0-100 分）
  - 评级：优秀（80+）、良好（60-79）、需改善（<60）

- **皮肤类型展示**
  - 显示皮肤类型标签
  - 提供简单的护肤建议

- **各项指标详情**
  - 8 项核心指标卡片展示：
    - 💧 水分：充足/偏低/严重缺水
    - 🛢️ 油分：低/适中/偏高
    - 🌡️ 敏感度：不敏感/轻度敏感/高度敏感
    - 🔴 痘痘：几乎无/轻微/中度/严重
    - 🌀 皱纹：几乎无/轻微/中度/严重
    - 🟤 色斑：几乎无/轻微/中度/严重
    - ⚫ 毛孔：几乎无/轻微/中度/粗大
    - ⚫ 黑头：几乎无/轻微/中度/严重
  - 每项指标显示数值和评价
  - 颜色标识：绿色（优秀）、黄色（一般）、橙色/红色（需改善）

- **主要问题识别**
  - 根据分析结果识别主要皮肤问题
  - 标签形式展示（如"缺水"、"暗沉"、"毛孔粗大"等）

- **护肤建议**
  - 根据皮肤分析结果提供护肤建议
  - 针对性建议（补水、控油、抗衰老等）

### 3. 产品推荐功能

#### 3.1 智能推荐
- **基于皮肤分析的个性化推荐**
  - 根据皮肤类型推荐产品
  - 根据皮肤问题推荐产品（痘痘、色斑、皱纹等）
  - 根据水分/油分水平推荐产品
  - 根据敏感度推荐温和/强化产品

- **推荐结果展示**
  - 产品名称
  - 产品功效说明
  - 适合肤质
  - 使用方法

#### 3.2 推荐来源
- 基于豆包视觉模型的分析结果
- 内置护肤产品数据库
- 匹配算法自动推荐

### 4. 历史记录功能

#### 4.1 检测历史列表
- **时间线视图**
  - 按时间倒序显示所有检测记录

    - 检测日期和时间
    - 皮肤类型
    - 综合评分
    - 主要问题标签

- **日历视图**

  - 日历上有检测记录的日期高亮显示

#### 4.3 记录管理
- **删除记录**
  - 支持删除单条历史记录

- **批量管理**

#### 4.4 数据持久化

### 7. 数据存储与云服务

#### 7.1 云数据库
- **用户表（users）**
  - id：用户 ID
  - openid：微信 OpenID
  - phoneNumber：手机号
  - nickname：昵称
  - avatarUrl：头像 URL
  - detectionCount：检测次数
  - createdAt：创建时间
  - updatedAt：更新时间

- **皮肤分析历史表（skin_analysis_history）**
  - id：记录 ID
  - userId：用户 ID
  - imageUrl：皮肤图片 URL
  - skinType：皮肤类型
  - concerns：皮肤问题数组
  - moisture：水分值
  - oiliness：油分值
  - sensitivity：敏感度
  - acne：痘痘指数
  - wrinkles：皱纹指数
  - spots：色斑指数
  - pores：毛孔指数
  - blackheads：黑头指数
  - recommendations：护肤建议数组
  - createdAt：检测时间

#### 7.2 云存储
- **皮肤照片存储**
  - 上传皮肤分析照片到云存储
  - 自动生成访问 URL
  - 支持图片压缩优化

#### 7.3 云托管
- **后端服务部署**
  - NestJS 后端服务
  - 自动扩容
  - 健康检查
  - HTTPS 支持

### 10. API 接口列表

#### 10.1 用户相关
- `POST /api/user/login` - 微信登录
- `POST /api/user/login/phone` - 手机号登录
- `GET /api/user/:userId` - 获取用户信息
- `PUT /api/user/:userId` - 更新用户信息
- `POST /api/user/:userId/increment-detection` - 增加检测次数

#### 10.2 皮肤分析相关
- `POST /api/skin/analyze` - 皮肤分析（上传图片）
- `GET /api/skin/recommend` - 获取产品推荐

#### 10.3 历史记录相关
- `GET /api/skin/history` - 获取历史记录
- `POST /api/skin/history` - 保存历史记录
- `DELETE /api/skin/history/:id` - 删除历史记录

#### 10.4 系统相关
- `GET /api` - 健康检查

## 🎯 用户使用流程

### 首次使用流程
1. 打开小程序，进入首页
2. 点击"我的"进行微信登录
3. 点击"开始检测"

7. 查看检测结果
8. 查看产品推荐
9. 返回首页，查看历史记录

### 日常使用流程
1. 打开小程序，点击"开始检测"
2. 进行皮肤检测
3. 查看结果和推荐
4. 在"历史"页面查看之前的记录
5. 对比不同时间的皮肤状态
