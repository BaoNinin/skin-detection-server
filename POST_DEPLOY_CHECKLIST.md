# 部署成功后快速检查清单

## ✅ 快速配置（5 步）

### 1️⃣ 获取服务地址
```
云开发控制台 > 云托管 > 服务 > 访问地址
复制：https://skin-detection-server-xxx.tcb.qcloud.la
```

### 2️⃣ 配置小程序域名
```
小程序后台 > 开发 > 开发设置 > 服务器域名

添加：
request 合法域名：https://<服务地址>
uploadFile 合法域名：https://<服务地址>
downloadFile 合法域名：https://<服务地址>
```

### 3️⃣ 修改 .env.local
```bash
PROJECT_DOMAIN=https://<服务地址>
```

### 4️⃣ 重新编译
```bash
pnpm build:weapp
```

### 5️⃣ 测试接口
```bash
# 浏览器访问
https://<服务地址>/api

# 预期响应
{"message":"API is running","timestamp":1234567890}
```
