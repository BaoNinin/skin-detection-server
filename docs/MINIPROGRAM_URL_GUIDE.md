2. 获取体验版二维码
3. 用户扫码即可体验

- ✅ 功能完整
- ✅ 可以分享给用户测试
- ❌ 需要审核通过

## 3. 🔗 小程序 URL Scheme

### 什么是 URL Scheme？

URL Scheme 是一种特殊的链接格式，可以从外部（如短信、邮件、浏览器）直接打开小程序。

### 获取方式

#### 方式 1：通过微信公众平台生成

1. 登录微信公众平台
2. 进入"开发" → "开发管理" → "开发设置"
3. 找到"URL Scheme"
4. 点击"获取 URL Scheme"
5. 配置参数：
   - 跳转路径：如 `pages/index/index`
   - 查询参数：如 `from=nfc&deviceId=001`
6. 生成 URL Scheme

#### 方式 2：通过 API 生成

```javascript
// 使用微信 API 生成
const res = await Taro.cloud.getURLScheme({
  cloudPath: 'pages/index/index',
  query: {
    from: 'nfc',
    deviceId: '001'
  }
})

console.log(res.openlink) // weixin://dl/business/?t=TICKET
```

### URL Scheme 格式
```
weixin://dl/business/?t=TICKET
```

**特点**：
- ✅ 可以从外部打开小程序
- ✅ 支持跳转到指定页面
- ✅ 可以传递参数
- ✅ 适合 NFC 芯片、二维码等场景

## 4. 📦 小程序码

### 获取方式

#### 方式 1：通过微信公众平台

1. 进入"设置" → "基本设置"
2. 点击"获取小程序码"
3. 选择小程序码类型
4. 下载小程序码图片

#### 方式 2：通过 API 生成

```javascript
// 使用微信 API 生成小程序码
const res = await Taro.cloud.openapi.wxacode.get({
  path: 'pages/index/index',
  width: 430
})

console.log(res.buffer) // 小程序码图片的 Buffer
```

### 小程序码 vs 二维码

| 特性 | 小程序码 | 二维码 |
|-----|---------|--------|
| 扫描速度 | 更快 | 较慢 |
| 容错率 | 更高 | 较低 |
| 美观性 | 更好 | 一般 |
| 圆形设计 | 支持 | 不支持 |

## 📝 注意事项

1. **AppID 配置**
   - 已配置 AppID：`wx8826c7b681ec3c65`
   - 需要在微信公众平台获取真实 AppID

2. **域名配置**
   - 小程序需要配置服务器域名白名单
   - 在微信公众平台 → 开发 → 开发管理 → 开发设置

3. **HTTPS 要求**
   - 生产环境必须使用 HTTPS
   - 开发环境可以使用 HTTP

4. **审核时间**
   - 小程序审核通常需要 1-7 天
   - 体验版可以绕过审核
